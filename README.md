commandlinefu.cn
================

打造 中文版的 http://commandlinefu.com

贡献你的力量
------------

 * 下载代码

```
$ git clone https://github.com/tg123/commandlinefu.cn.git
cd commandlinefu.cn
```

 * 寻找一个没有翻译的文件 

```
./tools/translateone.sh
```

 * 发送你的 `pull request`
 

实现原理
--------

网站使用 (jekyll)[http://jekyllrb.com/] 构建.
可以通过运行

```
jekyll build
```

在本地构建你的网站
